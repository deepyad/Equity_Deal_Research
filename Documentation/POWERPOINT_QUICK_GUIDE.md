## Quick Formatting Tips

### Slide Layouts to Use

- **Title Slide**: Use "Title Slide" layout
- **Content Slides**: Use "Title and Content" layout
- **Two-Column**: Use "Two Content" layout for comparison slides
- **Image-Heavy**: Use "Blank" layout, add text boxes manually

### Recommended Font Sizes

- **Title**: 44-48 pt (Bold)
- **Subtitle**: 36-40 pt
- **Bullet Points**: 18-24 pt
- **Body Text**: 16-20 pt
- **Captions**: 12-14 pt

### Color Scheme Suggestions

**Professional Blue Theme**:
- Primary: #1E3A8A (Dark Blue)
- Accent: #3B82F6 (Blue)
- Success: #10B981 (Green)
- Warning: #F59E0B (Orange)
- Background: #FFFFFF (White) or #F9FAFB (Light Gray)

### Visual Elements to Add

#### Icons
- Use PowerPoint's built-in icons (Insert â†’ Icons)
- Or download from: Flaticon, Icons8, Font Awesome
- Keep icon style consistent

#### Diagrams
- Use SmartArt (Insert â†’ SmartArt) for:
  - Process flows
  - Organizational charts
  - Cycles
- Use Shapes (Insert â†’ Shapes) for:
  - Custom diagrams
  - Flowcharts
  - Arrows and connectors

#### Charts
- Use Insert â†’ Chart for:
  - Bar charts (Performance metrics)
  - Pie charts (Score breakdowns)
  - Line charts (Timeline/roadmap)

## Common Issues & Solutions

### Issue: Too Much Text on Slide
**Solution**: 
- Use 6x6 rule (max 6 bullets, max 6 words per bullet)
- Break into multiple slides
- Use visuals instead of text

### Issue: Unreadable Fonts
**Solution**:
- Increase font size
- Use high contrast (dark text on light background)
- Test from back of room

### Issue: Inconsistent Design
**Solution**:
- Use Slide Master for consistency
- Create a style guide
- Reuse same shapes/icons

### Issue: Boring Slides
**Solution**:
- Add relevant images
- Use charts/graphs
- Include real examples
- Add company branding

## Presentation Mode Tips

1. Check all slides load properly
2. Test animations/transitions
3. Verify external links work
4. Prepare backup (PDF version)

1. Use Presenter View (F5, then Alt+F5)
2. Keep eye contact with audience

1. Share PDF version with attendees
2. Provide contact information
