# UI Screenshots Directory

This directory should contain screenshots of the Deal Similarity System user interface.

## Required Screenshots

Please add the following screenshots to this directory:

1. **main_search_interface.png** - Full page view showing the main search interface with sidebar, search form, and results area

2. **search_results.png** - Search results display showing multiple similar deals with similarity scores, breakdowns, and feedback buttons

3. **add_deal_form.png** - Add Deal tab showing the form for adding new deals with filled example data

4. **browse_deals.png** - Browse Deals tab showing filters (sector, deal type, year) and deal listings

5. **similarity_breakdown.png** - Detailed similarity score breakdown for a single deal showing structured, text, and metadata similarity components

## Taking Screenshots

1. Ensure the Streamlit app is running at `http://localhost:8501` (or your port)
2. Use your system's screenshot tool:
   - **Windows**: Win + Shift + S (Snipping Tool) or Win + PrtScn
   - **Mac**: Cmd + Shift + 4
   - **Linux**: Various tools available
3. Capture each interface element clearly
4. Save screenshots with the names listed above
5. Use PNG format for best quality

## Screenshot Guidelines

- **Resolution**: At least 1920x1080 or higher
- **Format**: PNG preferred (better quality), JPG acceptable
- **Content**: Show actual data/example deals for realism
- **Clarity**: Ensure text is readable, UI elements are visible
- **Completeness**: Capture entire relevant sections, not just parts

Once screenshots are added, update the consolidated document to reference them.
